## 2. 核心技术挑战与解决方案

### 2.1 挑战 1: 建立可靠的追溯映射

**问题**: OpenCASCADE 解析 STEP 文件后，如何保持与原始实体的对应关系？

**解决方案**: 使用 OCAF (Open CASCADE Application Framework)

```python
from OCC.Core.TDocStd import TDocStd_Document
from OCC.Core.XCAFDoc import XCAFDoc_ShapeTool
from OCC.Core.STEPCAFControl import STEPCAFControl_Reader

# 使用 XCAF 读取 STEP 文件（保留标签信息）
reader = STEPCAFControl_Reader()
reader.ReadFile("model.step")
reader.Transfer(doc)

# 获取 Shape Tool（包含标签映射）
shape_tool = XCAFDoc_ShapeTool.GetShapeTool(doc.Main())
```

### 2.3 挑战 3: 显示原始 STEP/IGES 文本

**问题**: 如何从实体 ID 快速定位到文件中的文本位置？

**解决方案**: 预处理建立实体 ID 到行号的索引

```python
class STEPFileParser:
    def __init__(self, filepath):
        self.entity_index = {}  # entity_id -> (line_number, text)
        self._build_index()
    
    def _build_index(self):
        with open(self.filepath, 'r') as f:
            for line_num, line in enumerate(f, 1):
                if line.startswith('#'):
                    entity_id = int(line.split('=')[0].strip('#'))
                    self.entity_index[entity_id] = (line_num, line.strip())
```
